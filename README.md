## What it is
Validoon is a **local-only** security signal classifier that produces explicit, auditable decisions.

## Core guarantees
- No uploads
- No network calls
- No tracking
- Fully deterministic execution

## Verdicts
- **ALLOW**
- **WARN**
- **BLOCK**

Each verdict includes:
- Severity score
- Confidence level
- Entropy signal
